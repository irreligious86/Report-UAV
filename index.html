    Про застосунок

    Звіт БПЛА — веб-додаток для швидкого формування структурованих звітів місій БПЛА безпосередньо в браузері. Створений для польового використання з телефону чи планшета. Працює офлайн після першого завантаження. Усі дані зберігаються виключно на вашому пристрої — нічого не передається на сервери.

    Форма звіту

    Головний екран додатку. Дозволяє сформувати повний текст звіту місії БПЛА.

    Поля форми:

    • Екіпаж — позивний (текст) + лічильник вильотів (1–25). Лічильник зберігається між сесіями і автоматично збільшується після кожного звіту.
    • Дата — дата місії.
    • Борт — вибір зі списку. Утримуйте для вільного вводу будь-якого значення.
    • Характер — тип місії. Вибір або вільний ввод (утримати).
    • Час зльоту — ручний ввод або кнопка «Зараз» для миттєвого запису поточного часу.
    • Час ураження/втрати — аналогічно, з кнопкою «Зараз».
    • Координати — формат MGRS: префікс + easting (5 цифр) + northing (5 цифр). Префікс обирається зі списку.
    • Боєприпас — вибір або вільний ввод.
    • Стрім — посилання на відеопотік або «---».
    • Результат — вибір або вільний ввод.

    Кнопка «Готово»: формує структурований текст звіту → автоматично копіює його в буфер обміну → зберігає в журнал → збільшує лічильник екіпажу.

    Журнал та статистика

    Екран перегляду збережених звітів та аналітики за обраний період.

    Фільтрація:

    • Діапазон дат (від / до) і часу. За замовчуванням — поточний місяць.
    • Фільтр працює за полем «Час ураження/втрати» зі звіту.
    • Цей же фільтр автоматично застосовується і до карти.

    KPI-панель (зверху):

    • Вильотів — загальна кількість місій за період.
    • Уражень — кількість результатів типу «Ураження».
    • Втрат — кількість результатів типу «Втрата».
    • Ефективність — відсоток уражень від загальної кількості вильотів.

    Вкладка «Статистика»:

    Зведення за період: кількість по бортах, боєприпасах, типах місій та результатах. Результати автоматично нормалізуються (наприклад, «Ураження цілі», «Ураження укриття» зараховуються як одна категорія «Ураження»). Кнопка Copy summary копіює зведення.

    Вкладка «Журнал»:

    Список звітів картками. Пошук по тексту (фільтрація в реальному часі). Для кожного звіту — кнопки Copy і Share. Кнопка Copy all копіює всі звіти одним блоком.

    Передача даних (Експорт / Імпорт)

    Секція внизу екрану журналу. Дозволяє переносити історію звітів між пристроями через зашифрований файл.

    Експорт:

    1. Натисніть «Експорт».
    2. Введіть ключ шифрування → підтвердіть його повторно.
    3. Завантажиться файл uav_reports.enc.json — зашифрований AES-256-GCM з ключем, виведеним через PBKDF2 (SHA-256, 250 000 ітерацій).
    4. Зберігайте файл і ключ окремо.

    Імпорт:

    1. Натисніть «Імпорт» → виберіть .json файл.
    2. Введіть ключ шифрування.
    3. Додаток перевірить версію файлу, цілісність даних і кожен запис.
    4. Звіти з файлу зіллються з існуючою історією без дублікатів (дедуплікація за часом + текстом).
    5. Журнал і карта оновляться автоматично.

    Карта місій

    Інтерактивна карта (OpenStreetMap + Leaflet) з маркерами координат MGRS зі звітів обраного періоду.

    Маркери:

    • Одна місія в точці — компактна цианова точка.
    • 2–3 місії — зелений круг з числом.
    • 4–5 місій — лаймовий круг з числом.
    • 6+ місій — великий янтарно-оранжевий круг з числом.

    Перегляд:

    • Клік по одиночній точці → overlay-карточка з повним текстом звіту, кнопками «Копіювати» та «Поділитися».
    • Клік по груповому маркеру → список місій (відсортований за часом, найновіші зверху). Клік по місії → карточка звіту. Кнопка «Назад» повертає до списку. Кнопка «✕» закриває все.

    Налаштування списків

    Редактор значень, які використовуються у випадаючих списках форми звіту:

    • Дрони
    • Типи місій
    • Боєприпаси
    • Результати
    • Префікси MGRS
    • Стріми

    Значення можна додавати, видаляти та перетягувати для зміни порядку. Зміни зберігаються на пристрої й мають пріоритет над базовою конфігурацією (config.json). Кнопка скидання повертає значення за замовчуванням.

    Навігація

    Довге натискання на заголовок «Звіт по БПЛА» у верхній частині будь-якого екрану відкриває меню переходу. Доступні екрани: Форма звіту, Журнал та статистика, Карта, Налаштування списків, Довідка та контакти. Активний екран підсвічується. «Закрити» — закрити меню.

    Встановлення PWA на смартфон

    Навіщо встановлювати:
    • швидкий запуск з іконки на головному екрані, без відкриття браузера вручну;
    • стабільна робота офлайн після першого завантаження;
    • менше шансів випадково закрити вкладку під час роботи;
    • зручніше використовувати в полі як окремий робочий інструмент.

    Android (Chrome):
    1) відкрийте застосунок у браузері Chrome;
    2) натисніть меню браузера (⋮);
    3) оберіть «Встановити застосунок» або «Додати на головний екран»;
    4) підтвердіть встановлення.
    Після цього «Звіт БПЛА» з'явиться як окрема іконка на екрані смартфона.

    iPhone / iPad (Safari):
    1) відкрийте застосунок у Safari;
    2) натисніть кнопку «Поділитися»;
    3) оберіть «На екран “Додому”»;
    4) підтвердіть додавання.
    Після цього застосунок запускається з іконки як окремий веб-додаток.

    Важливо: перший запуск має бути з інтернетом, щоб браузер закешував файли. Далі додаток працює офлайн.

    Конфіденційність та безпека

    • Додаток не збирає, не передає і не зберігає дані на зовнішніх серверах.
    • Історія звітів, лічильник, налаштування — тільки у localStorage вашого браузера.
    • Експортований файл зашифрований AES-256-GCM. Ключ шифрування ніде не зберігається — його вводить користувач.
    • Офлайн-режим: після першого завантаження застосунок повністю працює без інтернету.
